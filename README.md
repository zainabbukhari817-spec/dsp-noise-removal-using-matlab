# DSP Noise Removal Using MATLAB

## Overview
This project demonstrates audio noise removal using Digital Signal Processing (DSP) techniques in MATLAB. A Moving Average Filter is applied to reduce unwanted noise from an input audio signal while preserving the original audio quality.

## Features
- Reads an input audio file
- Applies a Moving Average Filter for noise reduction
- Displays original and filtered audio waveforms
- Saves the filtered audio as a new file
- Compares original and filtered signals

## Technologies Used
- MATLAB
- Digital Signal Processing (DSP)

## Project Files
- `audio_noise_removal.m` – MATLAB source code
- `audio.wav.m4a` – Original input audio
- `filtered_audio.wav` – Filtered output audio
- `output.png` – Comparison of original and filtered waveforms

## How to Run
1. Open the project in MATLAB.
2. Make sure `audio.wav.m4a` is in the same folder as the MATLAB script.
3. Run `audio_noise_removal.m`.
4. The filtered audio file and waveform plots will be generated automatically.

## Output
The project generates:
- Original audio waveform
- Filtered audio waveform
- Filtered audio file (`filtered_audio.wav`)
